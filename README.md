# todoList-clone
original: https://github.com/dhdbstjr98/todo_list

## 환경
Node.js v10.16.0  
Ubuntu 18.04

## HOW TO INSTALL
### 1. Clone this repository
```bash
git clone git@github.com:JJuOn/todoList-clone.git
```
```bash
cd todoList-clone
```
### 2. Install modules
```bash
npm install
```
### 3. Start app.js
```bash
node app.js
```
